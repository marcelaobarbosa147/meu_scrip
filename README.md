Explicação para executar o script:

Ao abrir o terminal dê o comando:  ./ executar.sh  
Desta forma o script da calculadora iniciará

Explicação do Código Python da Calculadora: Essa Calculadora além do que foi ensinado a fazer, eu tomei a iniciativa de criar uma opção a mais, alem das basicas.
A Opção de "potência" onde é possivel elevar um número ao outro, tipo 2 elevado a 3 que dá 8.
Usei o modo de "Calculadora infinita" onde após o resultado, o scrip entra em loop e inicia novamente.
Tudo usando linguagem de Python
